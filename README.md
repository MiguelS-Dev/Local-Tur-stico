🧩 Explicando o Desafio

Esse projeto foi criado como um desafio proposto no curso da Rocketseat, em que eu precisava construir uma página que representasse um local turístico usando apenas HTML e CSS, treinando principalmente a estruturação de código e o envio correto ao GitHub.

O objetivo principal foi:

✔️ Reproduzir uma interface estática

Seguindo um layout (possivelmente vindo de um modelo ou um design de referência), organizei a página para mostrar as informações de um local turístico — como nome, descrição, imagens ou qualquer outro conteúdo relacionado.

✔️ Trabalhar boas práticas de front-end

Durante o desenvolvimento, foquei em:

Utilizar estrutura semântica no HTML;
Estilizar com CSS puro;
Organizar os arquivos (HTML, CSS e assets) de forma clara;
Verificar o funcionamento visual da página em diferentes tamanhos de tela.
✔️ Treinar o fluxo com Git e GitHub

Além de desenvolver o front-end, esse desafio também me ensinou a:

Fazer o controle de versão com Git;
Commitar mudanças com mensagens claras;
Subir o projeto pro GitHub corretamente — seguindo boas práticas de versionamento e histórico.
🎯 O que eu aprendi com esse desafio

Ao final, consegui:

Construir uma página estática com layout organizado e visual coeso;
Fortalecer conceitos de HTML e CSS no desenvolvimento web;
Entender melhor como enviar e manter um projeto no GitHub;
Preparar essa base para futuros projetos mais complexos (como tornar a página responsiva, adicionar interatividade com JavaScript ou transformar em um projeto dinâmico).
